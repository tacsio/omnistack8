#  OmniStack by [@Rocketseat][rocketseat] ![rocketlogo][rocketlogo]

![applogo][applogo]

> Backend

```
$1 docker run -p 27017:27017 -it mongo
$2 yarn install
$2 yarn dev
```

> Frontend

```
yarn install
yarn start
```

> Mobile

```
yarn install
adb reverse tcp:3333 tcp:3333
yarn react-native run-android
```

***

```
$1 react-native start
$2 react-native run-android
```
